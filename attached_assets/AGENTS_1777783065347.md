# Budget Monitor — AGENTS.md

> **Purpose:** Autonomous agent execution guide. Follow every task in sequence. Do not skip phases. Do not proceed to the next task until the current task passes its verification step.  
> **Stack:** Laravel 11 · SQLite · Blade · Tailwind CSS · Alpine.js  
> **Working Directory:** Project root (where `artisan` lives)  
> **Agent Rule:** When a verification step fails, fix the issue before moving on. Never comment out failing code to make tests pass.

### TASK 01 — Create Laravel Project & Configure SQLite

**What to do:**
